# TimeTrack Pro

## Description

**English:**  
TimeTrack Pro is a Python-based attendance and salary management system built using Tkinter and pandas. It allows organizations to upload Excel attendance sheets, edit employee in/out times, calculate total monthly working hours, and compute salaries based on hourly rates. The system also accounts for weekends and holidays in salary calculation. Results can be saved as Excel or PDF files.

**German:**  
TimeTrack Pro ist ein Python-basiertes System zur Verwaltung von Anwesenheit und Gehalt, entwickelt mit Tkinter und pandas. Es ermöglicht Organisationen, Excel-Anwesenheitsblätter hochzuladen, Ein-/Aus-Zeiten der Mitarbeiter zu bearbeiten, die gesamten monatlichen Arbeitsstunden zu berechnen und Gehälter basierend auf Stundenlöhnen zu ermitteln. Das System berücksichtigt auch Wochenenden und Feiertage bei der Gehaltsberechnung. Ergebnisse können als Excel- oder PDF-Dateien gespeichert werden.

---


 ## Funktionen

- Excel-Anwesenheitsdateien hochladen und bearbeiten
- Gesamte monatliche Arbeitsstunden berechnen
- Gehälter basierend auf Stundenlöhnen kalkulieren
- Wochenenden und Feiertage berücksichtigen
- Ergebnisse als Excel oder PDF speichern
- Mitarbeiterdetails hinzufügen, bearbeiten und anzeigen

## Voraussetzungen

- Python 3.x
- pandas (pip install pandas)
- openpyxl (pip install openpyxl)
- Tkinter (in der Regel in Python enthalten)
- reportlab (optional, für PDF-Export: pip install reportlab)

Programm starten:
